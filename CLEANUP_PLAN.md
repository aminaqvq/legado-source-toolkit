**Do not delete automatically** — review and remove manually.

- `./cli.ts` → now at `./src/cli.ts`
- `./index.ts` → now at `./src/index.ts`
- `./constants/defaults.ts` → now at `./src/constants/defaults.ts`
- `./constants/keywords.ts` → now at `./src/constants/keywords.ts`

## Nested project (tests moved)

- `./legado-source-toolkit/` → tests moved to `./tests/`, can be deleted entirely

- `./dist/` — TypeScript build output (regenerated by `pnpm build`)

## How to clean

```bash
# Remove legacy root files
rm cli.ts index.ts
rm -rf constants/

# Remove nested project
rm -rf legado-source-toolkit/

# Remove build artifacts
rm -rf dist/ reports/
```
